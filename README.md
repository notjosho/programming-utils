## Quick Start Workflow

### New Python Project
```bash
mkdir myproject && cd myproject
git init
python3 -m venv venv
source venv/bin/activate
touch main.py .gitignore README.md
git add .
git commit -m "Initial commit"
```

### File Structure
```
project/
├── src/
│   ├── __init__.py
│   └── main.py
├── tests/
├── .gitignore
├── README.md
└── requirements.txt
```
